HTML Answers
  1. HTML stands for Hypertext Markup Language, and CSS stands for Cascading Style Sheets.
  2. HTML would be used to put paragraphs of text on a webpage.
  3. CSS would be used to change the font and background color of a button.
  4. An HTML tag is a code element that is used to describe the structure and content of a web page.
  5. The three parts of an HTML element are the opening tag, the content, and the closing tag.
  6. The purpose of the doctype declaration is to specify the version of HTML being used in the document.
  7. The HTML element is the root element of an HTML document and is used to encapsulate all other elements.
  8. The purpose of the head element is to provide metadata about the document, such as the document's title, keywords, and scripts.
  9. The purpose of the body element is to contain the main content of the document, such as text, images, and links.
  10. To create a paragraph in HTML, use the <p> tag.
  11. To create a heading in HTML, use the <h1>through <h6> tags.
  12. There are six levels of headings in HTML, with <h1> being the highest level and most important, and <h6> being the lowest level and least important.
  13. To make text bold and important in HTML, use the <strong> or <b> tag.
  14. To make text italicized and add emphasis to it in HTML, use the <em> or <i> tag.
  15. An element has a parent-child relationship with any nested elements within it.
  16. Two elements that are at the same level of nesting have a sibling relationship.
  17. To create HTML comments, use the <!-- comment --> syntax.
  18. The <ul> tag is used to create an unordered list in HTML.
  19. The <ol> tag is used to create an ordered list in HTML.
  20. The <li> tag is used to create list items within both unordered and ordered lists in HTML.
  21. The <a> tag is used to create links in HTML.
  22. An attribute is a modifier of an HTML element that provides additional information or instructions about that element.
  23. The href attribute tells links where to go.
  24. An absolute link is a link that includes the full URL of the linked page, while a relative link is a link that includes only the path of the linked page relative to the current page.
  25. The <img> element is used to display an image in HTML.
  26. The two attributes that images always need to have in HTML are the src attribute, which specifies the source URL of the image, and the alt attribute, which provides a text alternative for the image.
  27. To access a parent directory in a file path, use "../".
  28. The four main image formats that you can use for images on the web are JPEG, PNG, GIF and SVG.
